# Deidentified sample data

These CSVs are **synthetic/deidentified** examples used by the Source Integration guide.

- `athena-source-empi-input-sample.csv` — Example EMPI input records sourced from Athena.
- `caremgmt-source-empi-input-sample.csv` — Example EMPI input records from a **non-Athena** source (Care Management).
- `empi-crosswalk-gold-sample.csv` — Example rows from the final EMPI crosswalk showing **both** sources sharing the same `EMPI_ID` when a match is made.
